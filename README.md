# Voice Testing Platform

A comprehensive platform for testing voice agent communications and interactions.

## Features

- Agent-to-Agent voice communication testing
- WebSocket-based real-time audio streaming
- Twilio media stream integration
- Scaled testing capabilities
- Rich logging and monitoring
- Database-backed test suite management

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```

## API Endpoints

- `/v1/test-suites` - Test suite management
- `/v1/target-agents` - Target agent configuration
- `/v1/user-agents` - User agent configuration
- `/v1/test-cases` - Test case management
- `/v1/test-history` - Test run history and results
